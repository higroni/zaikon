# MASTER_CONFIG

This document is the golden copy for configuration keys.

## Application

```env
ZAIKON_ENV=development
ZAIKON_LOG_LEVEL=INFO
ZAIKON_API_HOST=127.0.0.1
ZAIKON_API_PORT=8100
```

## Storage

```env
ZAIKON_DATABASE_URL=sqlite:///./zaikon.db
ZAIKON_ARTIFACT_DIR=./data/artifacts
ZAIKON_UPLOAD_DIR=./data/uploads
```
